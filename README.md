# jpa-query-builder

## 프로그래밍 요구 사항

- 모든 로직에 단위 테스트를 구현한다. 단, 테스트하기 어려운 UI 로직은 제외
- 자바 코드 컨벤션을 지키면서 프로그래밍한다.
- 객체지향설계의 규칙을 다 지켜본다.
