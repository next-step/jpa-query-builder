    - [x] @Column의 nullable 속성을 사용한다.
    - [x] @Column의 nullable 속성이 선언되어 있지 않을 경우, true를 사용한다.
  - [x] Column의 길이를 설정한다.
    - [x] @Column의 length 속성을 사용한다.
